### wrk

Modern HTTP benchmarking tool

https://github.com/wg/wrk



```
make -j4

cp /tmp/wrk/wrk /usr/local/bin
```



```
wrk -t12 -c400 -d30s http://127.0.0.1:8080/index.html
```

### salvo

Boom的后继者

https://github.com/tarekziade/salvo

```
pip install salvo
```



```
salvo http://localhost:5000 -c 10 -n 100
```
